#  Store Inventory Application

A full-stack application built with **Java (Maven)** on the backend and **React** on the frontend, integrated with a **MongoDB Atlas** non-relational database.

The project demonstrates a simple end-to-end flow: fetching data from MongoDB through a REST API and displaying it in a modern web interface.

---


### Backend
- Java  
- Maven  
- Spring Boot  
- REST API  
- MongoDB Atlas (NoSQL)

### Frontend
- React  
- JavaScript  
- HTML / CSS  

### Database
- MongoDB Atlas (cloud-hosted NoSQL database)

---

## Features

-  Connection to MongoDB Atlas
-  GET endpoint for retrieving data from the database
-  RESTful backend architecture
-  React-based UI for visualizing stored data
-  Full-stack integration (Backend → Database → Frontend)

---

#  Functionality Overview

1. The backend exposes a **GET endpoint** that retrieves documents from MongoDB Atlas.
2. The frontend consumes this endpoint.
3. The UI displays the database content in a user-friendly format.

This provides a simple example of how a Java backend, a React frontend, and a NoSQL database can work together.
